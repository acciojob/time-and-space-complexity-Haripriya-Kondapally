Time Complexity and Space Complexity

Space Complexity and Time Complexity are two fundamental concepts in computer science used to evaluate the efficiency of algorithms. The time complexity of an algorithm quantifies the amount of time taken by an algorithm to run as a function of the length of the input. Note that the time to run is a function of the length of the input and not the actual execution time of the machine on which the algorithm is running on.

Problem-solving using computer requires memory to hold temporary data or final result while the program is in execution. The amount of memory required by the algorithm to solve given problem is called space complexity of the algorithm. O(n) – linear complexity – takes space directly proportional to the input size
